## Meetup VR

An example combining [A-Frame VR](https://aframe.io) with React and Redux. Based on [aframe-react-boilerplate](https://github.com/ngokevin/aframe-react-boilerplate).

### Getting Started

To get started:

```bash
npm install
npm run serve-js &
npm run serve
```

Then head over to localhost:5555.
